Personal Projects

Recently, I was taught a certain method of interpolation given a limited set of data to extrapolate points from an unknown function. This method is called Newtonian Interpolation. I used a method called ivided difference to calculate the polynomial's coefficients. I then used Shor's algorithm to evaluate the polynomial for a very large number of values. I created a user friendly application in java that asks the user how many points they will enter, whether they want to graph the function or look up a specific value, and then returns n data points specified by the user in a .dat file in whatever directory they choose. The user can then use a free graphing software such as gnuplot to visualize the points. The polynomial generated can be compared to a previously known function or can be used to approximate values outside a certain domain.

On the left, we have a comparison of a sixth degree polynomial to the know function, 1/(25t^2+1).
